# Aliyot / עליות

::: prayer
מִי שֶׁבֵּרַךְ אֲבותֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקב הוּא יְבָרֵךְ אֶת  [שֵׁם] בֶּן/בַּת [שֵׁם הָאָב]
בַּעֲבוּר שֶׁעָלָה לַתּורָה לִכְבוד הַמָּקום לִכְבוד הַתּורָה

::: prayer-shabbat
בשבת:וְלִכְבוד הַשַּׁבָּת
:::

::: prayer-chaggim
ביו"ט: וְלִכְבוד הַרֶגֶל
:::

::: prayer-high-holidays
בר"ה ויו"כ: וְלִכְבוד יום הַדִּין 
:::

בִשכַר זֶה הַקָּדושׁ בָּרוּךְ הוּא יִשְׁמְרֵהוּ וְיַצִּילֵהוּ מִכָּל צָרָה וְצוּקָה וּמִכָּל נֶגַע וּמַחֲלָה וְיִשְׁלַח בְּרָכָה וְהַצְלָחָה בְּכָל מַעֲשה יָדָיו

::: prayer-chaggim
ביו"ט: וְיִזְכֶּה לַעֲלות לָרֶגֶל
:::

::: prayer-high-holidays
בר"ה ויו"כ: וִיִכְתְּבֵהוּ וְיַחְתְּמֵהוּ לְחַיִּים טובִים בְּיום הַדִּין הַזֶּה 
:::

עִם כָּל יִשרָאֵל אֶחָיו וְנאמַר אָמֵן
:::
