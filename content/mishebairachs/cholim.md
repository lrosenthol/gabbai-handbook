# Sick / חולים

## Men / זכר

::: prayer
מִי שֶׁבֵּרַךְ אֲבותֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקב משֶׁה וְאַהֲרן דָּוִד וּשְׁלמה הוּא יְבָרֵךְ אֶת הַחולִים    __________________________________  
יִתן צְדָקָה בַּעֲבוּרָם, בִּשכַר זֶה הַקָּדושׁ בָּרוּךְ הוּא יִמָלֵא רַחֲמִים עָלָיהֶם לְהַחֲלִימָם וּלְרַפְּאתָם וּלְהַחֲזִיקָם וּלְהַחֲיותָם, וְיִשְׁלַח להֶם מְהֵרָה רְפוּאָה שְׁלֵמָה מִן הַשָּׁמַיִם לִרְמַ"ח אֵבָרָיהֶם וּשְׁסָ"ה גִּידָיהֶם בְּתוךְ שְׁאָר חולֵי יִשרָאֵל, רְפוּאַת הַנֶּפֶשׁ וּרְפוּאַת הַגּוּף

::: prayer-shabbat
בשבת: שַׁבָּת הִיא מִלִזְּעוק וּרְפוּאָה קְרובָה לָבוא, 
:::

::: prayer-chaggim
ביו"ט: יום טוב הוא מִלְזּעוק וּרְפוּאָה קְרובָה לָבוא,
:::

::: prayer-shabbat-chaggim
בשבת וְיו"ט: שַׁבָּת וְיום טוב הֵם מִלִזְּעוק וּרְפוּאָה קְרובָה לָבוא, 
:::

הַשְׁתָּא בַּעֲגָלָא וּבִזְמַן קָרִיב. וְנאמַר אָמֵן:
:::
