# מי שברך לחולה אחד

## לזכר (for a male patient)

::: prayer
מִי שֶׁבֵּרַךְ אֲבוֹתֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקֹב מֹשֶׁה וְאַהֲרֹן דָּוִד וּשְׁלֹמֹה,
הוּא יְבָרֵךְ וְיִרְפָּא אֶת הַחוֹלֶה: *(פלוני בן פלונית)*

בַּעֲבוּר שֶׁאָנוּ מִתְפַּלְלִים לִשְׁלוֹמוֹ.
הָאֵל זֶה הַקָּדוֹשׁ בָּרוּךְ הוּא יְמַלֵּא רַחֲמִים עָלָיו,
לְהַחֲלִימוֹ וּלְרַפְּאוֹתוֹ וּלְהַחֲזִיקוֹ וּלְהַחֲיוֹתוֹ,
וְיִשְׁלַח לוֹ רְפוּאָה שְׁלֵמָה מִן הַשָּׁמַיִם לְרַמַ״ח אֵבָרָיו וּשְׁסָ״ה גִּידָיו,
בְּתוֹךְ שְׁאָר חוֹלֵי יִשְׂרָאֵל, רְפוּאַת הַנֶּפֶשׁ וּרְפוּאַת הַגּוּף,
:::

::: prayer-shabbat
שַׁבָּת הִיא מִלִּזְעֹק וּרְפוּאָה קְרוֹבָה לָבוֹא,
:::

::: prayer-chaggim
יוֹם טוֹב הוּא מִלִּזְעֹק וּרְפוּאָה קְרוֹבָה לָבוֹא,
:::

::: prayer
וְרַחֲמִים בַּעֲגָלָא וּבִזְמַן קָרִיב, וְנֹאמַר אָמֵן.
:::
