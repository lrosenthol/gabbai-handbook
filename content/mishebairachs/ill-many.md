# מי שברך לחולים רבים

## לזכרים (for male patients)

::: prayer
מִי שֶׁבֵּרַךְ אֲבוֹתֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקֹב מֹשֶׁה וְאַהֲרֹן דָּוִד וּשְׁלֹמֹה,
וְאֶת אִמּוֹתֵינוּ שָׂרָה רִבְקָה רָחֵל וְלֵאָה,
הוּא יְבָרֵךְ אֶת הַחוֹלִים: *(פלוני בן פלונית, ופלוני בן פלונית, וכו׳)*

בַּעֲבוּר שֶׁאָנוּ מִתְפַּלְלִים לִשְׁלוֹמָם.
הָאֵל זֶה הַקָּדוֹשׁ בָּרוּךְ הוּא יְמַלֵּא רַחֲמִים עֲלֵיהֶם,
לְהַחֲלִימָם וּלְרַפְּאוֹתָם וּלְהַחֲזִיקָם וּלְהַחֲיוֹתָם,
וְיִשְׁלַח לָהֶם רְפוּאָה שְׁלֵמָה מִן הַשָּׁמַיִם לְכָל אֵבָרֵיהֶם וּלְכָל גִּידֵיהֶם,
בְּתוֹךְ שְׁאָר חוֹלֵי יִשְׂרָאֵל, רְפוּאַת הַנֶּפֶשׁ וּרְפוּאַת הַגּוּף,
:::

::: prayer-shabbat
שַׁבָּת הִיא מִלִּזְעֹק וּרְפוּאָה קְרוֹבָה לָבוֹא,
:::

::: prayer-chaggim
יוֹם טוֹב הוּא מִלִּזְעֹק וּרְפוּאָה קְרוֹבָה לָבוֹא,
:::

::: prayer
וְרַחֲמִים בַּעֲגָלָא וּבִזְמַן קָרִיב, וְנֹאמַר אָמֵן.
:::
